# 🔥 HabitTrack Pro

**HabitTrack Pro** ist ein minimalistischer, cloud-basierter Habit-Tracker, der speziell für die Unterstützung bei Abstinenz und persönlicher Weiterentwicklung entwickelt wurde. Das Ziel: Einfachheit, Fokus und Privatsphäre.

## ✨ Features

- **🛡️ Authentifizierung:** Sicherer Login und Registrierung über Supabase Auth.
- **☁️ Cloud-Sync:** Deine Daten werden in Echtzeit mit der Supabase-Datenbank synchronisiert.
- **📊 Progress-Tracking:** Visualisierung deiner Erfolge durch Fortschrittsbalken und Wochenberechnungen.
- **📱 PWA-Ready:** Installiere die App direkt auf deinem Smartphone (iOS & Android).
- **🎨 Custom UI:** Dark-Mode Design mit Emoji-Support für jedes Habit.
- **🔄 Reset-Funktion:** Setze deine Habits zurück oder lösche sie unwiderruflich.

## 🚀 Tech Stack

- **Frontend:** React.js mit Vite
- **Datenbank & Auth:** Supabase
- **Styling:** Custom CSS mit Fokus auf Mobile-First & Media Queries
- **PWA:** `vite-plugin-pwa`
